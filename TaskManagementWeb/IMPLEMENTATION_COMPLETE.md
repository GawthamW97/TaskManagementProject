## 🚀 How to Use

### Step 1: Start API
```bash
cd TaskManagementApp
dotnet run
# Runs on http://localhost:5000
```

### Step 2: Start Web UI
```bash
cd TaskManagementWeb
dotnet run
# Runs on http://localhost:5001
```

### Step 3: Open Application
```
Browse to: http://localhost:5001
```

### Step 4: Authenticate
- Login to get JWT token
- Store in browser localStorage
- All API calls will use the token

### Step 5: Start Using
- Create projects
- Create tasks
- Add comments
- Manage team members
